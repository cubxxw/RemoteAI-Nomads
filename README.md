### 🏥 社保 / 医保怎么办？

<details open>
<summary>💊 <strong>灵活就业参保完整攻略</strong></summary>

数字游民/自由职业者可以「**灵活就业**」身份自己参保，这是脱离公司后维持社保连续的主要方式。

| 项目 | 说明 |
|------|------|
| **可参保险种** | 仅**养老 + 医疗**两项（无失业/工伤/生育） |
| **养老缴费比例** | 全国统一 20%（8% 入个人账户，12% 入统筹） |
| **缴费基数** | 当地社平工资的 60%–300%，可自选档位 |
| **办理渠道** | 当地人社 APP、支付宝/微信「市民中心-社保」、电子税务局、社保大厅 |
| **户籍限制** | 🎉 **2025 关键更新**：除北京外，全国多数城市已放开外地户籍以灵活就业身份参保 |

> ⚠️ **断缴影响**：
> - **医保**：断缴次月起停用，部分地区超 3 个月连续缴费年限清零
> - **养老**：不清零但拉低养老金、影响退休时间，可随时补续、年限累计
>
> 💡 **建议**：即使旅居海外，也保留国内医保缴纳，回国就医有保障。长期离境者可评估「城乡居民医保」（缴费更低）。

</details>

### 💴 个税与身份

<details>
<summary>📑 <strong>自由职业者报税提示</strong></summary>

- **收入性质**：自由职业收入通常按「**劳务报酬**」或「**经营所得**」（个体户/工作室）申报
- **成立工作室/个体户**：可开对公账户、开发票、合规结汇，适合收入稳定的接单者
- **境外税务居民**：长期海外停留超 183 天可能改变税务居民身份，涉及 CRS 信息交换，务必咨询专业税务顾问
- **建议**：个税口径因地而异，具体以属地税务机关为准，高收入者建议找专业代理

</details>
